使用数组生成 Tab 列表

```json
[
  {
    label: "Item 1",
    icon: Icon,
    value: 1,
    isActive: true
  },
  {
    label: "Item 2",
    icon: Icon,
    value: 2,
    isActive: false
  },
  {
    label: "Item 3",
    icon: Icon,
    value: 3,
    isActive: false
  }
]
```

* 省略 "label" 则仅显示图标
* 省略 "icon" 则仅显示文字
